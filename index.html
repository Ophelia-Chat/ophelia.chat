Requirements

To use Ophelia, you must provide valid API keys:

  • OpenAI key: https://platform.openai.com
  • Anthropic key: https://console.anthropic.com

Without a valid API key, the app will not be able to fetch AI responses.

Getting Started

  1. Clone the repository:
    git clone https://github.com/kroonen/ophelia.git
  2. Open in Xcode: Open the project and run on a simulator or device.
  3. Configure Keys: Enter your OpenAI or Anthropic API key in the Settings screen before chatting.
